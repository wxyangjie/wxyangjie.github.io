游戏设计与开发项目合集

Jun 3, 2019

1、微信小游戏《草龙戏珠》

这个项目是我和一位同学参加“高校微信小程序开发大赛”小游戏组的作品，核心玩法是讨论出来的，我负责程序和动画制作（使用的微信小游戏可视化工具进行开发），另外那名同学负责美术。

游戏的核心玩法是瞄准射击，玩家通过点击屏幕来控制发射的时机，使得“草龙”移动的安全位置。游戏采用的无尽模式，随着玩家分数的提升，游戏难度会通过增加阻碍或者调整数值等方式进行调控。

2、三维体感游戏Demo

在大二期间（2016），我和另外四个同样爱好游戏的同学组了一个小团队制作了这个体感游戏，游戏的玩法包括搜集、射击、跑酷、舞蹈等。由于没有经验，将这个游戏设计的比较乱，每一个关卡是一个独立的玩法，但是正因为这些不足，我发现游戏设计比我想象中要复杂太多，从而我开始了学习游戏设计相关的知识的探索。

**开发工具：**Unity、Kinect SDK。

**我的工作：**总体上我们五个人每人负责一个故事关卡的设计与开发，但是其中会互相协助和商讨，以保证游戏统一的风格。

  • 因为我有一点美术基础，在项目中负责了部分建模和贴图的制作，游戏中的场景素材基本来源于unity商店，人物模型由我们自己使用Adobe Fuse制作而成。
  • 精卫填海关卡由我独立设计和开发，期间小伙伴们会互相提出意见，进行相应的调整。
