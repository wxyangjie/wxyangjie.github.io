2、思考（不计分）：

  • 到达接收天线的电波的传播路径都有哪些形式？

  • [ ] 直射、反射和散射。

  • 它们会在接收天线处产生什么影响？

  • [ ] 在接收天线处产生相互干扰，使接收天线所接收到的场强矢量、振幅和相位随时间急剧变化，这使信号很不稳定。
